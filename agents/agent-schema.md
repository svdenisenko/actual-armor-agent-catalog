# Agent Specification Schema

Each agent folder is expected to include the following files:
- `agent.yaml`
- `bootstrap.md`
- `system.md`
- `instructions.md`
- `output-template.md`
- `knowledge.md`
- `tools.md`
- `tests.md`
- `changelog.md`

## Expected Metadata Fields (`agent.yaml`)
- `id`
- `name`
- `description`
- `visibility`
- `category`
- `target audience`
- `use case`
- `user benefit`
- `required tools`
- `safety notes`
- `readiness fields`
- `version`
