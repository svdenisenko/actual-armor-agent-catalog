# Agents Directory

The `agents/` directory contains customer-facing marketplace agent specifications.

## Expected Agent Contents
Each agent folder should include:
- metadata (`agent.yaml`)
- bootstrap instructions (`bootstrap.md`)
- system persona (`system.md`)
- operating instructions (`instructions.md`)
- output template (`output-template.md`)
- knowledge notes (`knowledge.md`)
- tools policy (`tools.md`)
- tests (`tests.md`)
- changelog (`changelog.md`)

Internal or backoffice assistants do not belong in this directory.
