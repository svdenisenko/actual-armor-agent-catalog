# Tools

## Required tools
- `http`: HTTP(S) URL-fetch tool for trusted repository spec loading only.
- `webSearch`: discover external sources for claim verification when needed.
- `webContent`: read specific sources and verify claims.

## Rules
- Do not use `webSearch` for repository spec loading.
- Do not rely on snippets alone for serious factual claims.
- For verification, cite direct clickable source URLs.
- For pure language/framing analysis of user-provided text, tools are optional.
