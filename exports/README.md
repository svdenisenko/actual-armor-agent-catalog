# Public Agent Export Staging

This directory is the staging area for OpenClaw export artifacts related to **public** Actual Armor marketplace agents.

## Current expected input

- `aa-public-agents-raw.json` (generated by OpenClaw)

## Purpose of this directory

Before generating any public catalog specs, run the import validation process to:

1. Confirm minimum required fields exist.
2. Exclude internal/backoffice assistants.
3. Tag sensitive agents for review.
4. Assign public visibility levels.
5. Apply default readiness statuses.
6. Remove disallowed private/internal claims and context.

## Validation-first workflow

1. Place raw export JSON in this directory.
2. Apply rules in `docs/import-rules.md`.
3. Follow execution checklist in `scripts/validate-agent-export.md`.
4. Produce a validated intermediate dataset (no public specs yet).

## Non-goal (for this phase)

- Do **not** generate final public agent catalog specs yet.
