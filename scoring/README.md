# Agent Scoring Context

This directory defines a **public-safe scoring context** for evaluating new Actual Armor marketplace agent candidates.

## Purpose

The scoring materials in `scoring/` are intended to:

- define how new Actual Armor marketplace agent candidates are scored
- help Product Assistant prioritize draft agents
- avoid duplicate or weak marketplace agents
- keep scoring conservative and evidence-aware

## Important Boundaries

Scoring is a prioritization aid only. It is **not** any of the following:

- publication approval
- QA success
- proof of market demand

Use this framework to decide whether a candidate should move into draft creation and further review, not to declare launch readiness.
