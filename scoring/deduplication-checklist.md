# Deduplication Checklist

Use this checklist before creating any new draft agent.

- [ ] check existing `agents/**`
- [ ] check `marketplace/**`
- [ ] check `status/**`
- [ ] compare target user
- [ ] compare job-to-be-done
- [ ] compare expected output
- [ ] compare required tools
- [ ] compare safety caveats

Decision:

- [ ] net-new agent
- [ ] extension of existing agent
- [ ] duplicate
- [ ] too broad / needs splitting
