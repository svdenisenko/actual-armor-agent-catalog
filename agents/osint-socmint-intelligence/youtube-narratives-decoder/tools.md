# Tools — YouTube Narratives Decoder

## Available tools
- `http`: HTTP(S) URL-fetch tool for repository spec loading only.
- `webSearch`: discover external sources when claim verification is requested.
- `webContent`: read specific sources and verify claims.

## Rules
- Do not use `webSearch` for repository spec loading.
- Do not rely on snippets alone for serious factual claims.
- Do not claim YouTube transcript extraction unless an actual transcript/source is available.
- For pure transcript/framing analysis, external tools are optional.
